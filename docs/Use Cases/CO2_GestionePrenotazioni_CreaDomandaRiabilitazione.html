KEY VALUE
Nome : effettuaDomandaRiabilitazione()
Riferimenti : Gestione Prenotazioni
Pre-condizioni :
  • Il valore di flag attali associati allo studente che sta effettuando la domanda deve essere maggiore del numero limite N di flag tollerabili
  • Il sistema deve conoscere la matricola dello studente che sta effettuando la richiesta. Per questo motivo lo studente dovrá giá essere autenticato
  • Deve esistere almeno un'istanza di Amministratore per potergli recapitare la domanda
Post-condizioni :
  • Viene creata un'istanza di Domanda Riabilitazione
  • Viene creata un'associazione tra la nuova Domanda Riabilitazione e lo Studente richiedente
  • Viene creata un'associazione tra la nuova Domanda e l'Amministratore alla quale é stata inviata quest'ultima
